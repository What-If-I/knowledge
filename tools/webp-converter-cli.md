# webp-converter-cli
The CommandLine tool for converting jpg or png files to webp, recursively.

## Run
This command will convert all images to .webp
```bash
npx webp-converter-cli -r
```
You can delete old images with
```bash
find . -not -name "*.webp" -delete
```
