# Postgres CLI with autocompletion and syntax highlighting

## Install
**pipx**
```bash
pipx install pgcli
```
**brew**
```bash
brew install pgcli
```
## Usage
```bash
$ pgcli [database_name]

or

$ pgcli postgresql://[user[:password]@][netloc][:port][/dbname][?extra=value[&other=other-value]]
```
